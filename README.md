นี่คืองานสำหรับสอบกลางภาคของรายวิชา web programing

## Get started

- clone this repo
- install dependencies with command bellow
  using `pnpm`

```
pnpm install
```

or using `npm`

```
npm install
```

- to run web on dev mode, run command bellow

```
pnpm dev
```

or using `npm`

```
pnpm run dev
```
